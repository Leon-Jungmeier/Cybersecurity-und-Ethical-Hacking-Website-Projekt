# Cybersecurity-und-Ethical-Hacking-Website-Projekt
Eine deutschsprachige Lern- und Informationswebsite zu IT-Sicherheit, Ethik und rechtlichen Grundlagen. Ziel ist es, Bewusstsein für Cyberrisiken zu schaffen und sichere Verhaltensweisen zu fördern – ohne illegale Inhalte oder Angriffsanleitungen.
